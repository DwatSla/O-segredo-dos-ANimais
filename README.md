# O Segredo dos Animais

Site desenvolvido em HTML e CSS inspirado no filme "O Segredo dos Animais".

## Sobre o Projeto

Este site apresenta informações sobre o filme, como sinopse, personagens e curiosidades.  
O objetivo do projeto é praticar estruturação com HTML e estilização com CSS.

## Tecnologias Utilizadas

- HTML5
- CSS3
